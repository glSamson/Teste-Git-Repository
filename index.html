// git --version "confirma se o git está instalado" // git init "Versiona o projeto no git" // git status "mostra os arquivos versionados ou não no projeto" // git add file "adiciona o arquio desejado no repositório" // git add . "Adiciona todos os arquivos não versionados ao repositório" // git commit -a -m "comentário" "Para informar informar o Git que o arquivo rastreado (trakerd) precisa ser salvo (committed)" -a: seleciona todos os arquivos à "comissão" -m: adiciona uma mensgem, como uma nota do que foi alterado naquela adição de código (savepoint) // git config --global user.email "" "Registra o email do usuário" // git config --global user.name "" "Registra o nome do usuário" // Omit --global "Torna usuário cadastrado neste projeto "particular", ou seja, apenas para este projeto" // git branch "Monstra as banchs (ramificações) e deixa em a que se encontra localizado atualmente" // git merge "nome do banch" "pega o código de um branch selecionado e aloca no branch em que você se encontra"
